# COOLWELL CRM

Sistema de gestión de partes técnicos para servicios de asistencia.

## Características
- Dashboard con estadísticas
- Gestión de partes técnicos
- Calendario de citas
- Gestión de técnicos y clientes
- Inventario de repuestos
- Reportes y análisis

## Archivos
- `Coolwell_V2.html` - Versión activa
- `BASE COOLWELL.html` - Versión de respaldo

## Uso
Abre `Coolwell_V2.html` en tu navegador.

Los datos se guardan automáticamente en localStorage.
